# Estimación del Proyecto

## Suposiciones de la estimación
- El proyecto será desarrollado por una sola persona.
- Se tienen conocimientos básicos de desarrollo web.
- El sistema incluirá funciones básicas y no será muy complejo.

## Desglose del trabajo y tiempo estimado

| Módulo                    | Horas estimadas |
|---------------------------|-----------------|
| Análisis y planeación     | 10              |
| Diseño del sistema        | 12              |
| Gestión de usuarios       | 15              |
| Gestión de alumnos        | 12              |
| Gestión de materias       | 10              |
| Gestión de calificaciones | 15              |
| Pruebas básicas           | 10              |
| **Total estimado**        | **84 horas**    |

## Nivel de incertidumbre
Esta estimación tiene un nivel de incertidumbre medio, ya que durante el desarrollo pueden aparecer cambios en los requisitos o problemas técnicos no previstos.

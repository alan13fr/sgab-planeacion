# Calidad del Software
La calidad del software es importante para que el sistema funcione bien y cumpla con lo que los usuarios esperan.

## Usabilidad
El sistema debe ser fácil de usar, incluso para personas que no saben mucho de tecnología. 
Si es claro y sencillo, habrá menos errores y será más rápido aprender a usarlo.

Si no es fácil de usar, los usuarios pueden equivocarse mucho, no querer usar el sistema o necesitar ayuda todo el tiempo.

## Seguridad
El sistema debe proteger la información académica. 
Cada usuario debe iniciar sesión y solo poder ver lo que le corresponde.

Si no hay buena seguridad, personas no autorizadas podrían entrar, cambiar calificaciones o ver información privada.

## Confiabilidad
El SGAB debe funcionar sin fallos y guardar la información correctamente.
Además, los datos deben estar disponibles cuando se necesiten.
Si el sistema falla seguido, se puede perder información y los usuarios dejarán de confiar en él.

# Descripción del Proyecto

## Objetivo del sistema
El Sistema de Gestión Académica Básico (SGAB) tiene como objetivo administrar información académica de la universidad, permitiendo el registro y consulta de alumnos, materias, calificaciones y usuarios del sistema.

## Usuarios principales
- Administradores académicos
- Docentes
- Alumnos

## Problema que resuelve
Actualmente, llevar la información académica de forma manual o en varios programas diferentes provoca errores, datos repetidos y pérdida de tiempo. El SGAB reúne toda la información en un solo sistema, lo que hace más fácil consultarla y administrarla.

## Suposiciones iniciales
- El sistema será utilizado desde un navegador web.
- Los usuarios contarán con acceso a internet.
- El sistema manejará un volumen moderado de usuarios.
- No se consideran integraciones con sistemas externos en esta etapa.

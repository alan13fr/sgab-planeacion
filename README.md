# Sistema de Gestión Académica Básico (SGAB)

## Integrante
- Christian Alan Flores Ramos

## Descripción
Este repositorio contiene la planeación inicial de un Sistema de Gestión Académica Básico (SGAB), desarrollada como parte de la actividad de Ingeniería de Software.

## Objetivo del repositorio
Documentar el análisis para el desarrollo de un sistema de software, abordando aspectos clave como calidad, estimación, riesgos y alcance, aplicando fundamentos de la Ingeniería de Software antes de la etapa de programación.
